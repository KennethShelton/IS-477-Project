# IS 477 Course Project: Air Quality and Health in NYC

**Team:** Kenneth Shelton & Tianqi Fu  
**Course:** IS 477, Fall 2025

## Project overview

This project investigates the relationship between air pollution and health outcomes in New York City, analyzing which pollutants are most prevalent, where they occur, and how many residents may be affected by pollution-related health conditions.

## Quick start

### Prerequisites

```powershell
pip install -r requirements.txt
```

### Run the pipeline

From the project root directory:

```powershell
# 1. Inspect raw data
python .\scripts\inspect.py

# 2. Import CSVs into SQLite
python .\scripts\load_data.py

# 3. Run basic cleaning
python .\scripts\clean_data.py
```

This will create:
- `data/nyc_air_health.db` (SQLite database)
- `data/processed/air_quality_clean.csv`
- `data/processed/NYNY_BenMAP_clean.csv`

## Data sources

- **NYC Air Quality**: https://catalog.data.gov/dataset/air-quality
- **EnviroAtlas BenMAP**: https://catalog.data.gov/dataset/enviroatlas-new-york-city-ny-benmap-results-by-block-group3

Checksums for raw files are in `data/raw/checksums.txt`.
